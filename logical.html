논리형 인공어

논리형 인공어(Logical Language)는 언어의 모호성을 제거하고 사고를 명확하게 표현하기 위해 설계된 언어입니다.

1. 프로그래밍 언어와의 관계

C언어, Python, Java 같은 프로그래밍 언어도 넓은 범위에서 보면 '인공어'의 범주에 속합니다. 인간과 컴퓨터 간의 정확한 소통을 위해 만들어졌기 때문입니다.


2. 로지반 (Lojban)

대표적인 논리 언어인 로지반은 문법적 모호성이 전혀 없도록 설계되었습니다. 컴퓨터가 파싱(Parsing)하기 쉬운 구조를 가지고 있습니다.

  • 특징 1: 문화적 중립성
  • 특징 2: 수학적 논리 구조
